# Description: This project includes:
- NextJS;
- Sanity CMS;
- Typescript;
- SSR;
- NextAuth;
- Login by Google account;
- Tailwind CSS;
- Responsive;
- Get, Post - Tweet, Comment methods;

---------------------------------------

To run this project follow next steps:

1. `npm install`
2. You need configure your own sanity tokens api keys in .env.local file.
3. `npm run dev`
